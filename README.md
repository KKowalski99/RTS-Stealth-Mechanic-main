# RTS-Stealth

Game features: 
 - Player mouse-based movement
 - Enemy with AI state machine with various states 
(Patrol, Chase, Attack, Look Around)
 - Player stealth movement ability
 - Player stealth backstab mechanic 
 - NPC highlights on mouse over

Project features: 
 - Scene manager
 - Event system
 - Assemblies
 - Debug tool
